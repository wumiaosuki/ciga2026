# Scripts

Project runtime scripts live here.

- `Framework/Events`: type-safe event bus.
- `Framework/StateMachine`: lightweight code-driven state machine.
- `Framework/Singletons`: inheritable MonoBehaviour singleton bases.
- `Framework/UI`: UI prefab loading, panel lifecycle and layer management.
- `Framework/Utilities`: small shared helpers.

Use the `Ciga2026.Framework` assembly from gameplay assemblies by default.
